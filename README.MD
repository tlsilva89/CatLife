# 🐾 CatLife — Gestão Inteligente para Pets

> Plataforma full-stack para saúde, bem-estar e organização da vida dos seus gatos.

O **CatLife** é um ecossistema completo desenvolvido para facilitar a rotina de tutores de gatos. Com uma interface moderna e intuitiva, a aplicação centraliza o histórico médico, financeiro e a agenda de vacinação, integrando tecnologias de ponta como **GraphQL**, **Hangfire** e **IA Generativa (Google Gemini)**.

---

## ✨ Funcionalidades

- 📋 Gestão completa de perfis de gatos (raça, cor, idade, castração)
- 🏥 Controle de consultas e agendamentos veterinários
- 💰 Registro e acompanhamento de despesas por categoria
- 🤖 Triagem inicial de sintomas com **Google Gemini AI**
- 🔔 Alertas automáticos via **Telegram**
- 🔐 Autenticação segura com **JWT** e criptografia **Argon2**

## 🛠️ Configuração e Instalação

### Pré-requisitos

- [.NET 10 SDK](https://dotnet.microsoft.com/)
- [Node.js 20+](https://nodejs.org/)
- [Docker](https://www.docker.com/)

---

### 1. Banco de Dados (Docker)

Suba o contêiner do PostgreSQL:

```bash
docker-compose up -d
```

---

### 2. Backend

Configure as variáveis no arquivo `appsettings.json`:

```json
{
  "Jwt": {
    "Secret": "SUA_CHAVE_FORTE_AQUI"
  },
  "Telegram": {
    "BotToken": "TOKEN_DO_BOTFATHER"
  },
  "Gemini": {
    "ApiKey": "SUA_CHAVE_DO_GOOGLE_AI_STUDIO"
  }
}
```

Execute a API:

```bash
cd backend/CatLife.API
dotnet run
```

A API estará disponível em `https://localhost:5001/graphql`.

---

### 3. Frontend

Instale as dependências e inicie o servidor de desenvolvimento:

```bash
cd frontend
npm install
npm start
```

A aplicação estará disponível em `http://localhost:4200`.

## 🗄️ Esquema do Banco de Dados

Execute os comandos abaixo para criar as tabelas no banco `catlife_db`:

```sql
-- 1. Tabela de Usuários
CREATE TABLE "Users" (
    "Id"           INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    "CreatedAt"    TIMESTAMP WITH TIME ZONE NOT NULL DEFAULT NOW(),
    "UpdatedAt"    TIMESTAMP WITH TIME ZONE,
    "Name"         TEXT NOT NULL,
    "Email"        TEXT NOT NULL,
    "PasswordHash" TEXT NOT NULL
);

-- 2. Tabela de Gatos
CREATE TABLE "Cats" (
    "Id"          INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    "CreatedAt"   TIMESTAMP WITH TIME ZONE NOT NULL DEFAULT NOW(),
    "UpdatedAt"   TIMESTAMP WITH TIME ZONE,
    "Name"        TEXT NOT NULL,
    "Breed"       TEXT,
    "Color"       TEXT,
    "BirthDate"   TIMESTAMP WITH TIME ZONE NOT NULL,
    "IsCastrated" BOOLEAN NOT NULL,
    "UserId"      INTEGER NOT NULL REFERENCES "Users"("Id") ON DELETE CASCADE
);

-- 3. Tabela de Vínculo com Telegram
CREATE TABLE "TelegramLinks" (
    "Id"        INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    "CreatedAt" TIMESTAMP WITH TIME ZONE NOT NULL DEFAULT NOW(),
    "UpdatedAt" TIMESTAMP WITH TIME ZONE,
    "UserId"    INTEGER NOT NULL UNIQUE REFERENCES "Users"("Id") ON DELETE CASCADE,
    "ChatId"    TEXT NOT NULL,
    "LinkedAt"  TIMESTAMP WITH TIME ZONE NOT NULL
);

-- 4. Tabela de Despesas
CREATE TABLE "Expenses" (
    "Id"        INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    "CreatedAt" TIMESTAMP WITH TIME ZONE NOT NULL DEFAULT NOW(),
    "UpdatedAt" TIMESTAMP WITH TIME ZONE,
    "CatId"     INTEGER NOT NULL REFERENCES "Cats"("Id") ON DELETE CASCADE,
    "Category"  TEXT NOT NULL,
    "Amount"    NUMERIC(18,2) NOT NULL,
    "Date"      TIMESTAMP WITH TIME ZONE NOT NULL
);

-- 5. Tabela de Agendamentos
CREATE TABLE "Appointments" (
    "Id"        INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    "CreatedAt" TIMESTAMP WITH TIME ZONE NOT NULL DEFAULT NOW(),
    "UpdatedAt" TIMESTAMP WITH TIME ZONE,
    "CatId"     INTEGER NOT NULL REFERENCES "Cats"("Id") ON DELETE CASCADE,
    "Type"      TEXT NOT NULL,
    "DateTime"  TIMESTAMP WITH TIME ZONE NOT NULL,
    "Location"  TEXT
);

-- 6. Tabela de Consultas IA
CREATE TABLE "AIConsults" (
    "Id"           INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    "CreatedAt"    TIMESTAMP WITH TIME ZONE NOT NULL DEFAULT NOW(),
    "UpdatedAt"    TIMESTAMP WITH TIME ZONE,
    "CatId"        INTEGER NOT NULL REFERENCES "Cats"("Id") ON DELETE CASCADE,
    "Symptoms"     TEXT NOT NULL,
    "UrgencyLevel" TEXT NOT NULL
);
```
